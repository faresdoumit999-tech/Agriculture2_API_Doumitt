# 🚜 نظام DOUMITT الزراعي المتكامل (SaaS Edition)

نظام سحابي متطور مصمم لمساعدة المزارعين السوريين على إدارة فواتيرهم، مصاريفهم، وتحليل أرباح المحاصيل بدقة وسهولة. يعتمد النظام على معمارية **Multi-Tenancy**، مما يسمح لكل مزارع بالحصول على حساب معزول وآمن تماماً.

## 🌟 المميزات (Features)
* **نظام حسابات مستقل**: كل مستخدم لديه قاعدة بياناته الخاصة المعزولة عبر JWT Authentication.
* **استيراد ذكي من إكسل**: معالجة تلقائية لملفات الفواتير مع تصحيح ذكي للتواريخ لتناسب النمط السوري.
* **تحليلات دقيقة**: تقارير مفصلة لكل محصول (أوزان، عمولات، صافي ربح).
* **تطبيق موبايل (PWA)**: يمكن تثبيت النظام على الشاشة الرئيسية للموبايل واستخدامه كتطبيق نظامي.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Backend**: FastAPI (Python).
* **Database**: PostgreSQL (Hosted on Railway).
* **Frontend**: HTML5, Tailwind CSS, JavaScript (ES6).
* **Deployment**: GitHub Pages & Railway.

## 📁 هيكلية المشروع (Project Structure)
بناءً على التنظيم الأخير للمستودع:
* `main.py`: منطق السيرفر والـ API.
* `models.py`: تعريف جداول قاعدة البيانات.
* `templates/`: واجهة المستخدم الأمامية.
* `requirements.txt`: المكتبات اللازمة للتشغيل.
